# TitleBar

The `TitleBar` component allows the user to interact with electrons api by closing, minimizing, or maximizing the window.
The component will change depending on whether or not the window is maximized, giving the user the option to maximize or unmaximize depending on the situation.

export const Template = (args) => ({
  Component: TitleBar,
  props: args,
});
